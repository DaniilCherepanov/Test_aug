1. Метод contains:
   1. Ввод: string="SkyPro", symbol=""
   2. Фр: True
   3. Ор: False
   4. Комментарий: Поиск пустой строки в тексте нелогичен. Следует возвращать False или  исключение.

2. Метод delete_symbol:
   1. Ввод: string="SkyPro", symbol=""
   2. Фр: строка остаётся без изменений, но может возникнуть некорректное поведение
   3. Ор: выбрасывать исключение ValueError или  возвращать исходную строку
   4. Комментарий: Неочевидное поведение при пустом символе.
